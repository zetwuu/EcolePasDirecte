⚡ 3 Étapes Simples

1️⃣ Créer un compte Firebase
Allez sur console.firebase.google.com
Créez un projet (gratuit!)

2️⃣ Copier la configuration
Dans Firebase → Paramètres → Général → Vos applications
Copiez le code JavaScript

3️⃣ Mettre à jour firebase-config.js
Ouvrez le fichier et remplacez tout par votre configuration
Sauvegardez (Ctrl+S)
