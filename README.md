# EcolePasDirecte

Un projet web pour l'école fictive "École Pas Directe" avec intégration Firebase pour la messagerie.

## Structure du projet

- `index.html` : Page de connexion principale
- `src/` : Code source
  - `css/` : Feuilles de style
  - `js/` : Scripts JavaScript
  - `pages/` : Pages HTML supplémentaires
- `assets/` : Ressources statiques
  - `images/` : Images et icônes
  - `build/` : Fichiers de build (APK, etc.)
- `docs/` : Documentation et guides

## Démarrage

Ouvrez `index.html` dans votre navigateur pour commencer.
